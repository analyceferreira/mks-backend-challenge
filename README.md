## Description

API for controll of book catalog.


## Tech

- NestJS
- TypeScript
- TypeORM
- Swegger for documentation
- Database SQLite

## Use API

https://book-catalog-challenge.herokuapp.com/api/books

## Use API - Documentation

## Clone project

```bash
$ git clone https://github.com/analyceferreira/book-catalog-api-challenge.git
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
